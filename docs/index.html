Documentation

Everything you need to know about Tomato syntax

üìÑ Basic Syntax

Selectors use name: with properties indented below

button:
  bg blue-500
  color white
  pad 2 4
  round lg

üì¶ Components

Define reusable styles with component and use

component btn:
  pad 2 4
  round lg
  pointer
  smooth

button:
  use btn
  bg blue-500

üéØ Pseudo States

Add hover:, focus:, active: as nested blocks

button:
  bg blue-500
  
  hover:
    bg blue-600
    shadow lg
  
  focus:
    ring 2px solid blue-400

üì± Responsive

Use @mobile:, @tablet:, @desktop:

hero:
  pad 16
  grid 3
  
  @mobile:
    pad 4
    grid 1
  
  @tablet:
    grid 2
