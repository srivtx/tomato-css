The syntax

Read like English. Compile to CSS.

Same component. Two ways to write it. Tomato is fewer lines, fewer colons, and far fewer decisions.

button.tom TOMATO 
# A reusable button component
component btn:
  pad 2 4
  round lg
  bold
  bg rose-500
  color white

button:
  use btn

  hover:
    bg rose-600
output.css 

                    

                

            

        


        
        

            

                

                    How it works
                    
Three steps. That's the whole thing.

                


                
    
                    
  1. 
                        01
                        
    Write
    
                        
    Create a .tom file. Use components, tokens, and human syntax. No new language to learn.
    
                    
    2. 
                    
  2. 
                        02
                        
    Compile
    
                        
    Run one command. Get standard CSS. No runtime, no build step, no magic.
    
                    
    3. 
                    
  3. 
                        03

                    Features
                    
Everything you need.
Nothing you don't.

                


                

                    

                        

                            
                        

                        
Components

                        
Define once with component btn:, use anywhere with use btn. Nest them. Compose them.

                    


                    

                        

                            
                        

                        
Tailwind colors

                        
All 22 palettes, every shade. bg rose-500, color slate-900. Built in, zero config.

                    


                    

                        

                            
                        

                        
Human syntax

                        
row spread not display: flex; justify-content: space-between. round lg not border-radius: 0.5rem.

                    


                    

                        

                            
                        

                        
Responsive & dark mode

                        
@mobile, @tablet, @desktop, @dark, @light. Built into the syntax.
