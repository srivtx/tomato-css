See the difference

Same button. Two ways to write it.

button.tom TOMATO 
component btn:
  pad 2 4
  round lg
  bold
  bg rose-500
  color white

button:
  use btn
  hover:
    bg rose-600
output.css CSS 
/* Same output, 12 lines */
.btn {
  padding: 0.5rem 1rem;
  border-radius: 0.5rem;
  font-weight: bold;
  background: #f43f5e;
  color: #fff;
}

button {
  padding: 0.5rem 1rem;
  border-radius: 0.5rem;
  font-weight: bold;
  background: #f43f5e;
  color: #fff;
}

button:hover {
  background: #e11d48;
}
